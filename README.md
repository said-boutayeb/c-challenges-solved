## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/said-boutayeb/c-challenges-solved.git

# Navigate to project
cd c-challenges-solved

# Explore a collection
cd 42-piscine-practice
cat README.md

# Compile any exercise
gcc -Wall -Wextra -Werror file.c -o program
./program
```
