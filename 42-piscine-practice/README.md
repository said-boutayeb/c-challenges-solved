## 🎯 About

The **42 Piscine** is an intensive programming bootcamp teaching C from the ground up. This collection represents my journey through carefully designed exercises.

**Key Features:**
- 🎓 Official 42 School curriculum
- 📈 Progressive difficulty levels
- 🔄 Multiple solution approaches
- ✅ 100% Norminette compliant

## 🛠️ Development Standards

<div align="center">

### Compilation Requirements

```bash
gcc -Wall -Wextra -Werror source.c -o program
```

### Quality Standards

| Standard | Requirement |
|:---------|:------------|
| ✅ **Compilation** | No warnings with strict flags |
| ✅ **Style** | 42 Norminette compliant |
| ✅ **Memory** | Leak-free, safe operations |

## 🚀 Quick Start

```bash
# Navigate to collection
cd 42-piscine-practice

# Choose a level
cd level_0

# Read detailed guide
cat README.md

# Compile with standards
gcc -Wall -Wextra -Werror exercise.c -o exercise

# Run
./exercise [args]
```
